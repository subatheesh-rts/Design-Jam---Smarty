ABOUT
♦♦♦

Smarty is a Artificial Intelligent home system which senses all our applicances and people inside home. Smarty plays an important role when user are inside home and out of home. Smarty automatically senses threat and send alerts to the user through smarty mobile app or wearable devices.

Smarty helps in monitoring health, power consumption, water consumption predicts the future usage. It alerts us when other peoples enters our home without our knowledge.

FEATURES
♦♦♦

Home Appliances Control
Smarty monitor and control all appliances in the home, so that it could conserve more power.
Resource Monitoring
Smarty keeps track of water and electricity consumption and also displays statistics when user prompt it.
Disaster Alerts
when Smarty predicts the disaster to be occured, It analyse and provides the safe path of evacuvation and time.
Security Alerts
Smarty alerts us incase of security issues like theft, leakages etc...

HOW IT WORKS
♦♦♦
Login
User should login with either Biometric or password to control the Smarty system.
User Configuration
User need to configure the system for the first time as per his/her wish to do things.
Analytics
Smarty system displays the statistic of each and every equipment to the user over a period of time.
Usability and Alerts
User can control the System anytime using VOICE command or UI . Smarty alerts user incase any emergencies.
